JS Dialogs Library

Here you can test a custom dialog library. It provides three types of dialog windows: alert, confirm and prompt. It works very similarly to the built-in dialog windows on Javascript. Below, you can see how each of these functions are called.

The 3 functions that calls each of the 3 dialog windows work as simply as descripted below on this page.

If you want to use this library, you can download the related files using the GitHub repository link located at the bottom of the page(click here to get there).

Alert

showAlert("Alert!")

Confirm

showConfirm("Click OK to confirm or Cancel to cancel: ")

Prompt

showPrompt("Text example")
